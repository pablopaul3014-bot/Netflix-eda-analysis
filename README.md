
# Netflix EDA Analysis

Exploratory Data Analysis (EDA) sobre contenido de Netflix como parte de un proyecto académico de Data Science.

## 📌 Objetivo
Analizar el catálogo de Netflix para identificar patrones y tendencias en:
- Tipos de contenido (películas / series)
- Duración
- Países de producción
- Años de lanzamiento

## 🛠️ Herramientas utilizadas
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## 📂 Archivos del proyecto
- `EjerNetflix_FINAL.ipynb`: análisis exploratorio completo
- `Netflix_OK.pptx (1).pdf`: presentación de resultados

## 📊 Conclusiones
El análisis permite observar cómo evolucionó el catálogo de Netflix y qué tipo de contenido predomina según región y años.
